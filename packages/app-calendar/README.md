# @tearleads/app-calendar

Calendar components for Tearleads.

## Installation

This package is part of the Tearleads monorepo and is not published independently.

## Usage

```typescript
import { CalendarView, useCalendarEvents } from '@tearleads/app-calendar';
```

## Development

```bash
# Build
pnpm --filter @tearleads/app-calendar build

# Test
pnpm --filter @tearleads/app-calendar test

# Test with coverage
pnpm --filter @tearleads/app-calendar test:coverage
```
