# @tearleads/window-manager

Floating window manager and components for desktop-like web applications.

## Installation

This package is part of the Tearleads monorepo and is not published independently.

## Usage

```typescript
import { WindowManager, useSidebarRefetch } from '@tearleads/window-manager';
```

## Development

```bash
# Build
pnpm --filter @tearleads/window-manager build

# Test
pnpm --filter @tearleads/window-manager test

# Test with coverage
pnpm --filter @tearleads/window-manager test:coverage
```
