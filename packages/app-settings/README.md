# @tearleads/app-settings

Settings components for Tearleads.

## Installation

This package is part of the Tearleads monorepo and is not published independently.

## Usage

```typescript
import { SettingsPanel, useSettings } from '@tearleads/app-settings';
```

## Development

```bash
# Build
pnpm --filter @tearleads/app-settings build

# Test
pnpm --filter @tearleads/app-settings test

# Test with coverage
pnpm --filter @tearleads/app-settings test:coverage
```
