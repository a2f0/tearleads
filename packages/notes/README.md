# @tearleads/notes

Notes management components for Tearleads.

## Installation

This package is part of the Tearleads monorepo and is not published independently.

## Usage

```typescript
import { NotesWindow, NotesProvider, useNotesContext } from '@tearleads/notes';
```

## Development

```bash
# Build
pnpm --filter @tearleads/notes build

# Test
pnpm --filter @tearleads/notes test

# Test with coverage
pnpm --filter @tearleads/notes test:coverage
```
