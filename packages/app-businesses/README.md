# @tearleads/app-businesses

Business identifier utilities for Tearleads.

## Installation

This package is part of the Tearleads monorepo and is not published independently.

## Usage

```typescript
import { validateEIN, formatEIN } from '@tearleads/app-businesses';
```

## Development

```bash
# Build
pnpm --filter @tearleads/app-businesses build

# Test
pnpm --filter @tearleads/app-businesses test

# Test with coverage
pnpm --filter @tearleads/app-businesses test:coverage
```
