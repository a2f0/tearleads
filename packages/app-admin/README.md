# @tearleads/app-admin

Admin pages and windows for Tearleads.

## Installation

This package is part of the Tearleads monorepo and is not published independently.

## Note

This package is source-consumed via Vite aliases and does not require a separate build step.

## Development

```bash
# Test
pnpm --filter @tearleads/app-admin test

# Test with coverage
pnpm --filter @tearleads/app-admin test:coverage
```
