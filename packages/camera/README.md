# @tearleads/camera

Camera capture components for Tearleads.

## Installation

This package is part of the Tearleads monorepo and is not published independently.

## Usage

```typescript
import { CameraCapture, useCameraStream } from '@tearleads/camera';
```

## Development

```bash
# Build
pnpm --filter @tearleads/camera build

# Test
pnpm --filter @tearleads/camera test

# Test with coverage
pnpm --filter @tearleads/camera test:coverage
```
