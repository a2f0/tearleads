# @tearleads/app-health

Health tracking primitives for Tearleads.

## Installation

This package is part of the Tearleads monorepo and is not published independently.

## Usage

```typescript
import { HealthMetric, useHealthData } from '@tearleads/app-health';
```

## Development

```bash
# Build
pnpm --filter @tearleads/app-health build

# Test
pnpm --filter @tearleads/app-health test

# Test with coverage
pnpm --filter @tearleads/app-health test:coverage
```
