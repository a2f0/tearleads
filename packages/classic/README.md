# @tearleads/classic

Classic tag-and-note workspace for Tearleads.

## Installation

This package is part of the Tearleads monorepo and is not published independently.

## Usage

```typescript
import { ClassicWorkspace, useClassicContext } from '@tearleads/classic';
```

## Development

```bash
# Build
pnpm --filter @tearleads/classic build

# Test
pnpm --filter @tearleads/classic test

# Test with coverage
pnpm --filter @tearleads/classic test:coverage
```
