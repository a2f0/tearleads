# @tearleads/app-notes

Notes management components for Tearleads.

## Installation

This package is part of the Tearleads monorepo and is not published independently.

## Usage

```typescript
import { NotesWindow, NotesProvider, useNotesContext } from '@tearleads/app-notes';
```

## Development

```bash
# Build
pnpm --filter @tearleads/app-notes build

# Test
pnpm --filter @tearleads/app-notes test

# Test with coverage
pnpm --filter @tearleads/app-notes test:coverage
```
