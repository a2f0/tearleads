# @rapid/client

React client application for Rapid, built with Vite.

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

This will start the development server on `http://localhost:3000`.

## Building

```bash
npm run build
```

This will build the application for production in the `dist` directory.

## Preview Production Build

```bash
npm run preview
```

## Features

- React 18
- TypeScript
- Vite for fast development and building
- Shared types from `@rapid/types`
- Shared utilities from `@rapid/utils`
- API proxy configured to connect to `@rapid/api` on port 5001
