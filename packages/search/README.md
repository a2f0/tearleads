# @tearleads/search

Core full-text search functionality for Tearleads.

## Installation

This package is part of the Tearleads monorepo and is not published independently.

## Usage

```typescript
import { SearchIndex, searchFiles, buildSearchIndex } from '@tearleads/search';
```

## Development

```bash
# Build
pnpm --filter @tearleads/search build

# Test
pnpm --filter @tearleads/search test

# Test with coverage
pnpm --filter @tearleads/search test:coverage
```
