# @tearleads/email

Email components for Tearleads.

## Installation

This package is part of the Tearleads monorepo and is not published independently.

## Usage

```typescript
import { EmailList, EmailDetail, useEmailContext } from '@tearleads/email';
```

## Development

```bash
# Build
pnpm --filter @tearleads/email build

# Test
pnpm --filter @tearleads/email test

# Test with coverage
pnpm --filter @tearleads/email test:coverage
```
