# @tearleads/help

Help pages and window for Tearleads.

## Installation

This package is part of the Tearleads monorepo and is not published independently.

## Note

This package is source-consumed via Vite aliases and does not require a separate build step.

## Development

```bash
# Test
pnpm --filter @tearleads/help test

# Test with coverage
pnpm --filter @tearleads/help test:coverage
```
