# @tearleads/app-classic

Classic tag-and-note workspace for Tearleads.

## Installation

This package is part of the Tearleads monorepo and is not published independently.

## Usage

```typescript
import { ClassicWorkspace, useClassicContext } from '@tearleads/app-classic';
```

## Development

```bash
# Build
pnpm --filter @tearleads/app-classic build

# Test
pnpm --filter @tearleads/app-classic test

# Test with coverage
pnpm --filter @tearleads/app-classic test:coverage
```
