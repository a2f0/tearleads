# @tearleads/app-contacts

Contacts management components for Tearleads.

## Installation

This package is part of the Tearleads monorepo and is not published independently.

## Usage

```typescript
import { ContactsList, ContactDetail, useContacts } from '@tearleads/app-contacts';
```

## Development

```bash
# Build
pnpm --filter @tearleads/app-contacts build

# Test
pnpm --filter @tearleads/app-contacts test

# Test with coverage
pnpm --filter @tearleads/app-contacts test:coverage
```
