# @tearleads/businesses

Business identifier utilities for Tearleads.

## Installation

This package is part of the Tearleads monorepo and is not published independently.

## Usage

```typescript
import { validateEIN, formatEIN } from '@tearleads/businesses';
```

## Development

```bash
# Build
pnpm --filter @tearleads/businesses build

# Test
pnpm --filter @tearleads/businesses test

# Test with coverage
pnpm --filter @tearleads/businesses test:coverage
```
