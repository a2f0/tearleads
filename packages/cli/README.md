# @tearleads/cli

Tearleads CLI for database management.

## Installation

```bash
npm install -g @tearleads/cli
```

## Usage

```bash
# Run CLI commands
pnpm --filter @tearleads/cli dev <command>
```

## Development

```bash
# Build
pnpm --filter @tearleads/cli build

# Development mode
pnpm --filter @tearleads/cli dev

# Test
pnpm --filter @tearleads/cli test

# Test with coverage
pnpm --filter @tearleads/cli test:coverage
```
