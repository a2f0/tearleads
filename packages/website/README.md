# @tearleads/website

Marketing website for Tearleads built with Astro.

## Installation

This package is part of the Tearleads monorepo and is not published independently.

## Development

```bash
# Start development server
pnpm --filter @tearleads/website dev

# Build
pnpm --filter @tearleads/website build

# Preview production build
pnpm --filter @tearleads/website preview

# Test
pnpm --filter @tearleads/website test

# Test with coverage
pnpm --filter @tearleads/website test:coverage

# Run E2E tests
pnpm --filter @tearleads/website test:e2e
```
