# @tearleads/terminal

Terminal UI components and command engine for Tearleads.

## Installation

This package is part of the Tearleads monorepo and is not published independently.

## Usage

```typescript
import { Terminal, CommandEngine, useTerminal } from '@tearleads/terminal';
```

## Development

```bash
# Build
pnpm --filter @tearleads/terminal build

# Test
pnpm --filter @tearleads/terminal test

# Test with coverage
pnpm --filter @tearleads/terminal test:coverage
```
