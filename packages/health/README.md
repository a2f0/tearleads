# @tearleads/health

Health tracking primitives for Tearleads.

## Installation

This package is part of the Tearleads monorepo and is not published independently.

## Usage

```typescript
import { HealthMetric, useHealthData } from '@tearleads/health';
```

## Development

```bash
# Build
pnpm --filter @tearleads/health build

# Test
pnpm --filter @tearleads/health test

# Test with coverage
pnpm --filter @tearleads/health test:coverage
```
