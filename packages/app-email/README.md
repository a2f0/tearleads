# @tearleads/app-email

Email components for Tearleads.

## Installation

This package is part of the Tearleads monorepo and is not published independently.

## Usage

```typescript
import { EmailList, EmailDetail, useEmailContext } from '@tearleads/app-email';
```

## Development

```bash
# Build
pnpm --filter @tearleads/app-email build

# Test
pnpm --filter @tearleads/app-email test

# Test with coverage
pnpm --filter @tearleads/app-email test:coverage
```
