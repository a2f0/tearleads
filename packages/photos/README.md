# @tearleads/photos

Photo management components for Tearleads.

## Installation

This package is part of the Tearleads monorepo and is not published independently.

## Usage

```typescript
import {
  PhotosWindow,
  PhotosUIProvider,
  usePhotosUIContext
} from '@tearleads/photos';
```

## Development

```bash
# Build
pnpm --filter @tearleads/photos build

# Test
pnpm --filter @tearleads/photos test

# Test with coverage
pnpm --filter @tearleads/photos test:coverage
```
