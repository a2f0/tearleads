# @tearleads/ai

AI chat components for Tearleads.

## Installation

This package is part of the Tearleads monorepo and is not published independently.

## Usage

```typescript
import { AIWindow, AIProvider } from '@tearleads/ai';
```

## Development

```bash
# Build
pnpm --filter @tearleads/ai build

# Test
pnpm --filter @tearleads/ai test

# Test with coverage
pnpm --filter @tearleads/ai test:coverage
```
