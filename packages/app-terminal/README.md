# @tearleads/app-terminal

Terminal UI components and command engine for Tearleads.

## Installation

This package is part of the Tearleads monorepo and is not published independently.

## Usage

```typescript
import { Terminal, CommandEngine, useTerminal } from '@tearleads/app-terminal';
```

## Development

```bash
# Build
pnpm --filter @tearleads/app-terminal build

# Test
pnpm --filter @tearleads/app-terminal test

# Test with coverage
pnpm --filter @tearleads/app-terminal test:coverage
```
