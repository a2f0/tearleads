# @tearleads/vehicles

Vehicle profile utilities for Tearleads.

## Installation

This package is part of the Tearleads monorepo and is not published independently.

## Usage

```typescript
import { ... } from '@tearleads/vehicles';
```

## Development

```bash
# Build
pnpm --filter @tearleads/vehicles build

# Test
pnpm --filter @tearleads/vehicles test

# Test with coverage
pnpm --filter @tearleads/vehicles test:coverage
```
