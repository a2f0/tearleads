# @tearleads/audio

Audio player components for Tearleads.

## Installation

This package is part of the Tearleads monorepo and is not published independently.

## Usage

```typescript
import { AudioWindow, AudioProvider, useAudioContext } from '@tearleads/audio';
```

## Development

```bash
# Build
pnpm --filter @tearleads/audio build

# Test
pnpm --filter @tearleads/audio test

# Test with coverage
pnpm --filter @tearleads/audio test:coverage
```
