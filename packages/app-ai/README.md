# @tearleads/app-ai

AI chat components for Tearleads.

## Installation

This package is part of the Tearleads monorepo and is not published independently.

## Usage

```typescript
import { AIWindow, AIProvider } from '@tearleads/app-ai';
```

## Development

```bash
# Build
pnpm --filter @tearleads/app-ai build

# Test
pnpm --filter @tearleads/app-ai test

# Test with coverage
pnpm --filter @tearleads/app-ai test:coverage
```
