# @tearleads/app-camera

Camera capture components for Tearleads.

## Installation

This package is part of the Tearleads monorepo and is not published independently.

## Usage

```typescript
import { CameraCapture, useCameraStream } from '@tearleads/app-camera';
```

## Development

```bash
# Build
pnpm --filter @tearleads/app-camera build

# Test
pnpm --filter @tearleads/app-camera test

# Test with coverage
pnpm --filter @tearleads/app-camera test:coverage
```
