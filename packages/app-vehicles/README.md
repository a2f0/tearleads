# @tearleads/app-vehicles

Vehicle profile utilities for Tearleads.

## Installation

This package is part of the Tearleads monorepo and is not published independently.

## Usage

```typescript
import {
  formatVehicleDisplayName,
  normalizeVehicleProfile,
  type VehicleProfile
} from '@tearleads/app-vehicles';
```

## Development

```bash
# Build
pnpm --filter @tearleads/app-vehicles build

# Test
pnpm --filter @tearleads/app-vehicles test

# Test with coverage
pnpm --filter @tearleads/app-vehicles test:coverage
```
