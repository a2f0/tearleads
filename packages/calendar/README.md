# @tearleads/calendar

Calendar components for Tearleads.

## Installation

This package is part of the Tearleads monorepo and is not published independently.

## Usage

```typescript
import { CalendarView, useCalendarEvents } from '@tearleads/calendar';
```

## Development

```bash
# Build
pnpm --filter @tearleads/calendar build

# Test
pnpm --filter @tearleads/calendar test

# Test with coverage
pnpm --filter @tearleads/calendar test:coverage
```
