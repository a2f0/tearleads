# @tearleads/app-search

Core full-text search functionality for Tearleads.

## Installation

This package is part of the Tearleads monorepo and is not published independently.

## Usage

```typescript
import { SearchIndex, searchFiles, buildSearchIndex } from '@tearleads/app-search';
```

## Development

```bash
# Build
pnpm --filter @tearleads/app-search build

# Test
pnpm --filter @tearleads/app-search test

# Test with coverage
pnpm --filter @tearleads/app-search test:coverage
```
