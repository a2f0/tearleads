# @tearleads/settings

Settings components for Tearleads.

## Installation

This package is part of the Tearleads monorepo and is not published independently.

## Usage

```typescript
import { SettingsPanel, useSettings } from '@tearleads/settings';
```

## Development

```bash
# Build
pnpm --filter @tearleads/settings build

# Test
pnpm --filter @tearleads/settings test

# Test with coverage
pnpm --filter @tearleads/settings test:coverage
```
