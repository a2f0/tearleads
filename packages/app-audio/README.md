# @tearleads/app-audio

Audio player components for Tearleads.

## Installation

This package is part of the Tearleads monorepo and is not published independently.

## Usage

```typescript
import { AudioWindow, AudioProvider, useAudioContext } from '@tearleads/app-audio';
```

## Development

```bash
# Build
pnpm --filter @tearleads/app-audio build

# Test
pnpm --filter @tearleads/app-audio test

# Test with coverage
pnpm --filter @tearleads/app-audio test:coverage
```
