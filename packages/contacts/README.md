# @tearleads/contacts

Contacts management components for Tearleads.

## Installation

This package is part of the Tearleads monorepo and is not published independently.

## Usage

```typescript
import { ContactsList, ContactDetail, useContacts } from '@tearleads/contacts';
```

## Development

```bash
# Build
pnpm --filter @tearleads/contacts build

# Test
pnpm --filter @tearleads/contacts test

# Test with coverage
pnpm --filter @tearleads/contacts test:coverage
```
